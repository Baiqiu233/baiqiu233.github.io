Windows IP 地址映射

  1. 打开C:\Windows\System32\drivers\etc\hosts
  2. 在最后添加your_IP_addr your_name
  3. 退出保存
    以后就可以用your_name代替your_IP_addr

重装系统后无法登录

删除曾经的登录信息：ssh-keygen -R+IP

配置免密登录

生成公钥：ssh-keygen

（MacOS）上传公钥：ssh-copy-id -i file_addr username@IP
（windows）上传公玥
1. ssh连接服务器
2. 编辑.ssh中的authorized_keys
3. 将你的公玥追加到字符后面
